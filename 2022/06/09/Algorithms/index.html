Algorithms

Array

2 Sum

  • Method: Hash Table, Time Complexity O(N)
  • Description: Traverse through the array, see if (target-i) exists in the hash table. If true, return these two indices; if false, store the current element and its index into the hash table.

Group Anagrams

  • Method: Hash Table, Time Complexity O(NKlog(K)); O(K*log(K)) to sort each string; O(N) for each element.
  • Description: Tuple of sorted string as keys, array element as values. Noted: In python, only immutable
    data structure (hashable) can be the keys.

Linked List

Definition

Summary

  • O(1) for insert or delete
  • O(n) for get (traverse)
  • Each element is a ListNode
  • Hash Table is useful to traverse the LinkedList and hold its node
  • Two pointers is useful to optimize the space complexity to O(1)
  • Dummy header is useful to simplify edge case (e.g: only contain one node)

Binary Tree

Definition

Maximum Depth of Binary Tree

  • Method: Recursion, DFS
  • Description: Each node’s maximum depth(MD) equals to the maximum of the MD of its sub-left tree and its sub-right tree
  • Method: Iteration, DFS
  • Description: Using stack to store the node and depth; go to the next level of tree can be simulated by stack pop node and append its subtree.

Invert Binary Tree

  • Method: Recursion, DFS
  • Description: Go one level deeper and convert the sub-left tree and sub-right tree

Subtree of Another Tree

  • Method: Recursion, DFS
  • Description: Define a helper function to tell if two trees are the same. Each time, go one level deeper to see if left/right subtree is the sametree.

Stack

Valid Parentheses

  • Method: Stack
  • Description: Traverse through the string, each time if it’s a left bracket, push to the stack. Each time meet a right bracket, pop the last left bracket and see if they’re match.
